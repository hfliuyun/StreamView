# Normative References For The First Milestone

StreamView records field names, versioned clause references, and original
project-authored explanations. Standards text is not committed to or
distributed with this repository.

| Area | Normative baseline |
|---|---|
| H.264 elementary-stream syntax | ITU-T Recommendation H.264 (08/2024) |
| AAC-LC and AudioSpecificConfig | ISO/IEC 14496-3:2019, Edition 5 |
| MPEG-4 Systems descriptors (`esds`) | ISO/IEC 14496-1:2010 |
| ISO Base Media File Format | ISO/IEC 14496-12:2026, Edition 8 |
| AVC carriage in ISO BMFF | ISO/IEC 14496-15:2024 with Amendment 1:2025 |

Prepublished or draft editions are not silently substituted. Updating a
baseline requires an ADR, a field-level compatibility review, and refreshed
conformance vectors.
