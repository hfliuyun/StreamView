# Mapping-Preserving Rule Transformations

The format definition language will allow bounded mapped transformations that forward, skip, and slice input while automatically preserving exact source mappings. Common transformations such as H.264 EBSP-to-RBSP conversion may be packaged as verified pure helpers, but remain expressible through the same constrained model. Rules may expose computed fields without source locations, but may not manufacture logical bits and present them as source-backed syntax fields.
